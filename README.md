# Programmer Calculator

Simple calculator for programmers. 

Supported bases:
*  Decimal (base 10)
*  Hex (base 16)
*  Binary (base 2)
*  Octal (base 8)

Operations:
*  Shift
*  Invert
*  Bitwise xor, and, or
*  Addition, subtraction
*  Multiplication, division, modulus
